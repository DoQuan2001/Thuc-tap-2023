## I. 1 VÀI KHÁI NIỆM.

### 1.1. USB BOOT LÀ GÌ?

 USB Boot chính là một thiết bị Boot hay còn được gọi là Boot Device. Nó tương tự như một ổ cứng để chứa hệ điều hành (OS) và khi quá trình Boot này diễn ra
